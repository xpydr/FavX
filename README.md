## Features

- 🤝 **Community-driven favours** — Request help or offer support to others nearby.
- 📝 **Quick favour posting** — Create requests with clear categories and rewards.
- 🔔 **Smart notifications** — Stay updated when favours are accepted, completed, or discussed.
- 🏆 **Leaderboard system** — Encourage participation through points and recognition.
- 🎁 **Redeemable rewards** — Turn kindness points into useful real-life perks.
- ⚙️ **Simple account settings** — Manage your profile, preferences, and notifications.

## Project Summary
The Favour Exchange web application is a reputation-based, peer-to-peer community platform for requesting and completing favours. This application aims to bring communities closer together by encouraging teamwork and mutual trust. 

## Project Vision
The vision of FavX is to create a platform where communities can support each other through acts of kindness and collaboration. By removing the monetary barrier and replacing it with a credit-based system, FavX encourages people to contribute to their local communities while building trust and meaningful connections.

## Project Requirements 
- 👤**User accounts** — Users must be able to register, log in, and manage their profiles.
- 📝 **Create favours** — Users can post requests describing the help they need.
- 🤝 **Accept favours** — Community members can volunteer to complete favours.
- 🏆 **Points system** — Users earn kindness points for helping others.
- 🎁 **Rewards marketplace** — Points can be redeemed for real-world perks.
- 🔔 **Activity notifications** — Users receive updates when favours are accepted or completed.
- 💬 **User communication** — Participants can interact and coordinate favours.
